# Misión Selección

## Escenario
Recursos Humanos necesita filtrar perfiles para un cargo de **Ingeniero Senior** de forma automática.

## El Problema
Para ser pre-seleccionado, el candidato debe cumplir al menos una de estas dos rutas:

1. Tener más de **5 años de experiencia** y dominar **Python**
2. Tener un título de **Maestría** y hablar **Inglés**

## Nota Crítica ⚠️
Si el candidato tiene **antecedentes de faltas éticas**, queda descartado **inmediatamente**, sin importar sus estudios o experiencia.

## Criterios de Preselección
- **Ruta 1**: 5+ años de experiencia Y domina Python
- **Ruta 2**: Título de Maestría Y habla Inglés
- **Descarte automático**: Antecedentes éticos negativos

## Reto para el Estudiante
Identificar el uso de operadores **Y (AND)**, **O (OR)** y el condicional de descarte prioritario.
