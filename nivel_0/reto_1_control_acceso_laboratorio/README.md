# El Control de Acceso al Laboratorio Químico

## Escenario
El laboratorio de alta seguridad tiene un protocolo de entrada estricto para evitar accidentes.

## El Problema
El sistema permite la entrada si el usuario tiene una identificación válida y porta su traje de protección. Sin embargo, si el sensor detecta que hay una fuga de gas en el interior, nadie puede entrar, sin importar su identificación o equipo.

## Reto para el estudiante
Determinar cuándo se imprime "Acceso Autorizado", "Acceso Denegado por Seguridad" o "Equipo Incompleto".
